* ======================================================================================
* Replication files for:
* Cieslak, A., A. Morse, and A. Vissing-Jorgensen: "Stock returns over the FOMC cycle," 
* Journal of Finance, forthcoming
* README
* ======================================================================================

* The input data is fomc_data_base.dta.

* The Stata code is process_final_post_01.do. 

* The code in the section "DATA PROCESSING/DEFINITIONS" constructs all variables needed and saves them in the dataset fomc_data1.dta.

* The code in the section "TABLES AND FIGURES" reproduces the tables and figures in the paper.

* Formatting of figures is controlled by FOMC.scheme.
